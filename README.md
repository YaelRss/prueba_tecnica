# Prueba Técnica - portal web jugueteria

Esta prueba tiene como enfoque crear un pequeño portal web que filtra contenido dependiendo el perfil del usuario

## Requerimientos cumplidos:
- Lenguajes: HTML, CSS, PHP 
- Base de Datos: MySQL 
- Control de versiones: Uso de GIT con historial de commits
- Funcionalidad: Filtro por género y envío de información por correo
